# AI Notes

## What is Artificial Intelligence (AI)?

Artificial Intelligence (AI) is technology that enables computers to simulate human cognitive functions. It enables machines to learn from data, recognize patterns, solve complex problems and make decisions.
Examples:
- Generative AI Chatbots
- Virtual Assistants
- Email Spam Filters

## What is Generative AI?

Generative AI is a type of artificial intelligence that creates new, original content such as text, images, code, audio or video by learning patterns from existing data.
Examples:
- ChatGPT
- Google Gemini
- Claude

## What is a Large Language Model (LLM)?

A Large Language Model (LLM) is a type of artificial intelligence built on deep neural networks designed to read, understand and generate human language.
Examples:
- GPT Models (OpenAI)
- Gemini (Google)
- Claude (Anthropic)

## What is Prompt Engineering?

Prompt Engineering is the process of designing effective prompts to obtain better responses from AI systems.

Example:
Basic Prompt:
Write a tournament announcement.

Improved Prompt:
Write an exciting [game] tournament announcement under 100 words.
Winner: Team 1
MVP: Player 1

## How AI Applications Work?

AI applications process user input, analyze it using trained AI models, and generate a relevant response based on learned patterns from data.
Workflow:

User → Prompt → AI Model → Response → User

## What is an API?

An API, or Application Programming Interface, is a set of defined rules and protocols that allow different software applications to communicate and exchange data with one another.

## What is an AI API?

An AI API is a software bridge that allows applications to access pre-trained AI models and their capabilities, without requiring the developers to build or train those complex algorithms from scratch.

## What is an API Key?

An API Key is a unique alphanumeric code used to identify and authenticate access to an API.

## What is JSON?

JSON (JavaScript Object Notation) is a text-based format used to store and transport structured data between systems.
Example:
```json
{
  "winner": "Team 1",
  "mvp": "Player 1"
}
```
